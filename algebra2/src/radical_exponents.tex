\section*{Instructions}
Solve each problem. Show clear work. Aim to finish in 60 minutes or less. Unless stated otherwise, assume variables are real; when simplifying even roots, use absolute value as needed to keep results nonnegative. When simplifying expressions, rewrite with positive exponents.

\section*{Problems}
\begin{enumerate}
  \item Simplify $(27x^{3}y^{-2})^{\tfrac{2}{3}}$ (assume $x>0$, $y>0$).
  \item Simplify $(16a^{4}b^{6})^{\tfrac{3}{4}}$; indicate where absolute values are required.
  \item Simplify $\dfrac{x^{3/2}y^{-1/2}}{x^{-1/2}y^{3/4}}$ (assume $x>0$, $y>0$).
  \item Simplify $\sqrt{25u^{2}v^{4}w^{3}}$, using absolute values where appropriate.
  \item Simplify $\sqrt{81t^{2}(t-4)^{4}}$, expressing any absolute values needed.
  \item Solve for $x \ge 0$: $x^{3/2} = 27$.
  \item Factor and solve $6x^{2} - 7x - 5 = 0$.
  \item Solve the inequality $2(3 - x) \le 5x + 1$.
  \item Factor completely: $x^{3} - 4x^{2} - x + 4$.
  \item Solve $4z^{2} + 4z + 7 = 0$ using the quadratic formula; give roots in $a + bi$ form.
  \item Simplify using exponent rules: $\left(5p^{-2}q^{1/2}\right)\left(\dfrac{p^{3}}{q}\right)$.
  \item Find the vertex of $y = -2x^{2} + 8x - 1$ and state whether it is a maximum or minimum; give the extreme value.
\end{enumerate}
